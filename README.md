# Nextjs-Learning
Learned the basics of Next.js project structure, including file organization and routing in the app directory. I built static and dynamic routes, added navigation, and used the Image component for optimization. I also explored server and client components, loading states, API routes, metadata, and basic SEO to improve performance and visibility.

This project is based on a tutorial and is created for learning and practice purposes only. It is not a production-ready project.
